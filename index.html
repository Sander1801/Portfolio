Cuts 'N Fades

Cuts 'N Fades is a barbershop located in Panningen. For the owner of the barbershop, I have voluntarily built a website on which customers can schedule appointments online.

The website is developed with the following programming and markup languages: PHP, jQuery, HTML, CSS, and Bootstrap.
A relational database management system is used to store data. As a software design pattern, I use a model-view-controller.
